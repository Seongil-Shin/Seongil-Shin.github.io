### **전위식**

연산자를 먼저 표시하고 연산에 필요한 피연산자를 나중에 표시

```
(A + B) * (C - D)
((A + B) * (C - D))
*(+(AB)-(CD))
*+AB-CD
```

<br/>

### **후위식**

피연산자를 먼저 표시하고, 연산자를 나중에 표시

```
(A+B) * (C-D)
((A+B) * (C-D))
((AB)+(CD)-)*
AB+CD-*
```
