MVVM

Model - View - View Model мқҳ м•ҪмһҗлЎң, н”„лЎңк·ёлһЁмқҳ л№„м§ҖлӢҲмҠӨ лЎңм§Ғкіј н”„л Ҳм  н…Ңмқҙм…ҳ лЎңм§Ғмқ„ UIлЎң лӘ…нҷ•н•ҳкІҢ л¶„лҰ¬н•ҳлҠ” нҢЁн„ҙ

кө¬м„ұмҡ”мҶҢ

  • Model : лҚ°мқҙн„°лҘј ліҙкҙҖн•ҳкі  мһҲлҠ” л¶Җл¶„мңјлЎң, лҚ°мқҙн„°лҘј л¶Ҳлҹ¬мҳӨкұ°лӮҳ м—…лҚ°мқҙнҠён•ҳлҠ” лЎңм§Ғмқҙ мһҲмқҢ.
  • View Model : Modelм—җ лҚ°мқҙн„°лҘј мҡ”мІӯн•ҳкі  к°Җкіөн•Ё. л№„м§ҖлӢҲмҠӨ лЎңм§Ғмқ„ мІҳлҰ¬. ViewлЎңл¶Җн„° мһ…л Ҙмқ„ л°ӣм•„ м Ғм Ҳн•ң мІҳлҰ¬лҘј н•Ё
  • View : UI лӢҙлӢ№. UIм—җ м—°кҙҖлҗң лЎңм§Ғл§Ң мҲҳн–үн•ңлӢӨ. н•„мҡ”н•ң лҚ°мқҙн„°лҠ” View Modelкіј data bindingмқ„ нҶөн•ҙ м–»лҠ”лӢӨ. мӮ¬мҡ©мһҗмқҳ мһ…л Ҙмқ„ VMмңјлЎң м „лӢ¬.

нҠ№м§•

  • ViewмҷҖ ViewModelмқҙ n : 1 кҙҖкі„мқҙлӢӨ. н•ҳлӮҳмқҳ View Modelм—җ м—¬лҹ¬ Viewк°Җ кІ°н•©мқҙ к°ҖлҠҘн•ҳм—¬ View Modelмқҳ мһ¬мӮ¬мҡ©м„ұмқҙ лҶ’лӢӨ.

мһҘм җ

  • лҚ°мқҙн„°лҘј кҙҖлҰ¬н•ҳлҠ” лЎңм§Ғкіј UIлЎңм§Ғмқ„ к№”лҒ”н•ҳкІҢ л¶„лҰ¬н•ҳм—¬ мң м§ҖліҙмҲҳм—җ мҡ©мқҙн•ҙм§җ.

  • мҪ”л“ңмқҳ мһ¬мӮ¬мҡ©м„ұмқ„ к°ңм„ 
  • UI л””мһҗмқҙл„ҲмҷҖ к°ңл°ңмһҗк°Җ мүҪкІҢ нҳ‘м—…н•  мҲҳ мһҲлҠ” л””мһҗмқё нҢЁн„ҙ

лӢЁм җ

  • View Modelмқҳ м„Өкі„к°Җ мүҪм§Җ м•ҠлӢӨ.

MVP

Model - View - Prensenter

кө¬мЎ°

  • Model : MVVMкіј лҸҷмқј
  • View : MVVMкіј лҸҷмқј
  • Prensenter : Viewм—җм„ң мҡ”мІӯн•ң м •ліҙлЎң Modelмқ„ к°Җкіөн•ҳм—¬ Viewм—җ м „лӢ¬н•ҙмЈјлҠ” л¶Җл¶„. ViewмҷҖ Modelмқ„ л¶ҷм—¬мЈјлҠ” м ‘м°©м ң к°ҷмқҖ м—ӯн• мқҙлӢӨ.

лҸҷмһ‘

  1. мӮ¬мҡ©мһҗмқҳ Action л“ӨмқҖ ViewлҘј нҶөн•ҙ л“Өм–ҙмҳЁлӢӨ.
  2. ViewлҠ” лҚ°мқҙн„°лҘј presenterлЎң мҡ”мІӯн•ңлӢӨ.
  3. presenterлҠ” modelм—җкІҢ лҚ°мқҙн„°лҘј мҡ”мІӯн•ңлӢӨ.
  4. modelмқҖ prensenterм—җкІҢ лҚ°мқҙн„°лҘј мқ‘лӢөн•ңлӢӨ.
  5. presenterлҠ” viewм—җкІҢ лҚ°мқҙн„°лҘј мқ‘лӢөн•ңлӢӨ.
  6. viewлҠ” presenterк°Җ мқ‘лӢөн•ң лҚ°мқҙн„°лЎң нҷ”л©ҙмқ„ кҫёлҰ°лӢӨ.

нҠ№м§•

PresenterлҠ” ViewмҷҖ Modelмқҳ мқёмҠӨн„ҙмҠӨлҘј к°Җм§Җкі  мһҲм–ҙ л‘ҳмқ„ м—°кІ°н•ҳлҠ” м ‘м°©м ң к°ҷмқҖ м—ӯн• мқ„ н•ңлӢӨ.

лҳҗ, PrensenterмҷҖ ViewлҠ” 1:1 кҙҖкі„мқҙлӢӨ.

мһҘм җ

  • ViewмҷҖ Model мӮ¬мқҙмқҳ мқҳмЎҙм„ұмқҙ м—ҶлӢӨ.

лӢЁм җ

  • ViewмҷҖ Prensenter мӮ¬мқҙмқҳ мқҳмЎҙм„ұмқҙ лҶ’лӢӨ.

лӮҳлҠ” м–ҙл–»кІҢ н•ҳкі  мһҲм§Җ?

MVC, MVVM, MVP л“ұмқҳ л””мһҗмқё нҢЁн„ҙмқ„ кіөл¶Җн•ҳл©ҙм„ң к·ёлҸҷн•ң л¬ҙм§Җм„ұмңјлЎң к°ңл°ңн–ҲлҚҳ лӮҙ кіјкұ° н”„лЎңм қнҠёк°Җ кіјм—° м–ҙл–Ө л””мһҗмқёнҢЁн„ҙм—җ к°Җк№Ңмҡҙм§Җ к¶ҒкёҲн•ҙм ё н•ңлІҲ кіјкұ° н”„лЎңм қнҠёлҘј мӮҙнҺҙліҙкё°лЎң н–ҲлӢӨ.

мқҙкёҖмқ„ мһ‘м„ұн•ҳкі  мһҲлҠ” мӢңм җм—җм„ңмқҳ лӮҳмқҳ к°ҖмһҘ мөңк·ј н”„лЎңм қнҠёмқё yourlist-web-renewal н”„лЎңм қнҠёмқҳ л””л үн„°лҰ¬ кө¬мЎ°лҠ” лӢӨмқҢкіј к°ҷлӢӨ.

  • components : м»ҙнҸ¬л„ҢнҠё
    • elements : кіөнҶөмңјлЎң мӮ¬мҡ©н•ҳлҠ” мһ‘мқҖ м»ҙнҸ¬л„ҢнҠёл“Ө
    • [м»ҙнҸ¬л„ҢнҠёлӘ…] - мЈјлЎң нҺҳмқҙм§Җ лӢЁмң„лЎң лӮҳлүЁ
      • index.tsx : м»ҙнҸ¬л„ҢнҠё мөңмғҒлӢЁ
      • container : м»ҙнҸ¬л„ҢнҠёлҘј кө¬м„ұн•ҳлҠ” мһ‘мқҖ лӢЁмң„ м»ҙнҸ¬л„ҢнҠёл“Өмқҳ м»Ён…Ңмқҙл„Ҳ
      • view : м»ҙнҸ¬л„ҢнҠёлҘј кө¬м„ұн•ҳлҠ” мһ‘мқҖ лӢЁмң„ м»ҙнҸ¬л„ҢнҠёл“Өмқҳ л·°
      • elements : м»ҙнҸ¬л„ҢнҠё лӮҙм—җм„ң кіөнҶөмңјлЎң мһҗмЈј мӮ¬мҡ©лҗҳлҠ” мһ‘мқҖ м»ҙнҸ¬л„ҢнҠё
  • modules
    • index.ts : лӘЁл“Ҳ мөңмғҒлӢЁ
    • moduleTypes.ts : лӘЁл“Ҳм—җм„ң кіөнҶөмңјлЎң мӮ¬мҡ©лҗҳлҠ” нғҖмһ…л“Ө
    • [лӘЁл“ҲлӘ…]
      • index.ts : лӘЁл“Ҳ мөңмғҒлӢЁ. лҰ¬л“Җм„ң нҸ¬н•Ё
      • saga.ts : лӘЁл“Ҳмқҳ лҰ¬лҚ•мҠӨ мӮ¬к°Җ кҙҖл Ё мҪ”л“ң лӘЁмқҢ
      • action.ts : м•Ўм…ҳ нғҖмһ…, м•Ўм…ҳ мғқм„ұ н•ЁмҲҳ л“ұ нҸ¬н•Ё
      • types.ts : лӘЁл“Ҳм—җм„ң мӮ¬мҡ©лҗҳлҠ” нғҖмһ…л“Ө

к°„лӢЁнһҲ л””мһҗмқё нҢЁн„ҙкіј кҙҖл Ёмқҙ мһҲлҠ” л¶Җл¶„л§Ң м Ғм—ҲлӢӨ.

лЁјм Җ modulesлҘј мӮҙнҺҙліҙл©ҙ, м•ұм—җ н•„мҡ”н•ң лҚ°мқҙн„°лҘј к°Җм§Җкі  мһҲкі , к°Ғмў… apiмҷҖ м§Ғм ‘ мҶҢнҶөн•ҳм—¬ лҚ°мқҙн„°лҘј л°ӣм•„л“Өмқҙкі , к°„лӢЁн•ң к°Җкіөмқ„ лӢҙлӢ№н•ҳлҠ” л¶Җл¶„мқҙлӢҲ Model м—җ н•ҙлӢ№н•ңлӢӨкі  ліј мҲҳ мһҲлӢӨ.

к·ёлҰ¬кі  к°Ғ м»ҙнҸ¬л„ҢнҠёмқҳ мөңмғҒлӢЁм—җ мһҲлҠ” index.tsx лҠ” лӘЁлҚёкіј мҶҢнҶөн•ҳм—¬ лҚ°мқҙн„°лҘј л°ӣм•„л“Өмқҙкі , к·ё м»ҙнҸ¬л„ҢнҠём—җм„ң мӮ¬мҡ©лҗҳлҠ” л№„м§ҖлӢҲмҠӨ лЎңм§Ғмқ„ лӢҙлӢ№н•ңлӢӨ. л”°лқјм„ң мқҙлҠ” MVPмқҳ PrensenterлӮҳ MVVMмқҳ View Modelм—җ н•ҙлӢ№н•ңлӢӨкі  ліј мҲҳ мһҲлӢӨ.

container лҠ” Modelкіј лҢҖнҷ”н•ҳм§Җ м•Ҡкі , мһҗмӢ мқҙ лӢҙлӢ№н•ҳлҠ” viewмқҳ мқҙлІӨнҠёлҘј мІҳлҰ¬н•ҙмЈјкі , мһҗмӢ мқҙ лӢҙлӢ№н•ҳлҠ” л¶Җл¶„м—җм„ңмқҳ Modelкіј м—°кҙҖмқҙ м—ҶлҠ” л№„м§ҖлӢҲмҠӨ лЎңм§Ғмқ„ мІҳлҰ¬н•ңлӢӨ. л”°лқјм„ң мқҙлҠ” MVVM кө¬мЎ°мқҳ View controllerм—җ к°Җк№қлӢӨкі  ліҙмқёлӢӨ.

мқҙл•Ң лӮҙ н”„лЎңм қнҠём—җм„ң viewлҠ” containerлӮҳ indexлЎң лҚ°мқҙн„°лҘј мҡ”мІӯн•ҳм§Җ м•ҠлҠ”лӢӨ. мғҒнғңлҘј нҶөн•ҙ container лҳҗлҠ” indexлЎңл¶Җн„° лҚ°мқҙн„°лҘј л°ӣм•„м„ң к·ёкІғмқ„ нҷңмҡ©н•ҳм—¬ UIлҘј кө¬м„ұн•ҳкі , UIм—җ кҙҖл Ёлҗң лЎңм§Ғл§Ңмқ„ нҸ¬н•Ён•ҳкі  мһҲлӢӨ. мқҙлҠ” MVVM кө¬мЎ°м—җм„ңмқҳ Viewм—җ мўҖлҚ” к°Җк№қлӢӨ.

л”°лқјм„ң лӮҙ н”„лЎңм қнҠёлҠ” көімқҙ л”°м§Җмһҗл©ҙ, MVVM кө¬мЎ°лҘј л”°лҘҙкі  мһҲлӢӨкі  ліј мҲҳ мһҲлӢӨ.

н•ҳм§Җл§Ң MVVMмқҳ нҢЁн„ҙм—җм„ң лІ—м–ҙлӮң л¶Җл¶„лҸ„ мЎҙмһ¬н•ңлӢӨ.

  1. index.tsxлҠ” View Modelкіј ProviederлҘј лҸҷмӢңм—җ кІён•ҳкі  мһҲлӢӨ.
    • Modelкіј лҢҖнҷ”н•ҳл©° к°Ғмў… л№„м§ҖлӢҲмҠӨ лЎңм§Ғмқ„ мІҳлҰ¬н•ҳл©°(View Model), View ControllerмқҙлӮҳ ViewлЎң лҚ°мқҙн„°лҘј ліҙлӮҙмӨҖлӢӨ(Provider)
    • мқҙлҠ” к°Ғ м»ҙнҸ¬л„ҢнҠё лі„лЎң н•ҳлӮҳмқҳ кұ°лҢҖн•ң View Model + ProviderлҘј л§Ңл“ӨкІҢ н•ҳм—¬, View Modelкіј Viewкіј 1:nмңјлЎң м—°кІ°лҗҳм–ҙ, View Modelмқ„ мһ¬мӮ¬мҡ©н•  мҲҳ мһҲлӢӨлҠ” MVVMмқҳ мһҘм җмқ„ м—Ҷм•ӨлӢӨ.
    • лҳҗ view controllerмқҳ мһ‘м—…мқ„ лӢҙлӢ№н•ҳкі  мһҲлҠ” л¶Җл¶„лҸ„ ліҙмқёлӢӨ.
  2. л¶Җл¶„л¶Җл¶„м ҒмңјлЎң index.tsx, containerм—җм„ң uiлҘј лӢҙлӢ№н•ң л¶Җл¶„мқҙ мЎҙмһ¬н•ңлӢӨ
    • мҳҲлҘј л“Өм–ҙ, index.tsxм—җм„ң к°„лӢЁн•ң wrapperлҘј л‘ҳлҹ¬мӢём„ң viewлҘј мЎ°н•©н•ҳлҠ” л“ұмқҳ View мһ‘м—…мқ„ мЎ°кёҲ л§Ўм•„ н•ҳкі  мһҲлӢӨ.
    • мқҙлҠ” viewмқҳ мһ‘м—…мқ„ л¶„мӮ°мӢңнӮӨлҠ” н–үмң„лЎң, л””мһҗмқё ліҖкІҪмӮ¬н•ӯмқҙ мһҲмқ„ мӢң мҲҳм •мқ„ к№ҢлӢӨлЎӯкІҢ н•ҳкі  мһҲлӢӨ.

к°ңм„  мӮ¬н•ӯ
