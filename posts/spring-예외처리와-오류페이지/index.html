м„ңлё”лҰҝ мҳҲмҷёмІҳлҰ¬

м„ңлё”лҰҝмқҳ мҳҲмҷёмІҳлҰ¬ л°©мӢқ

Exception

WAS <- м„ңлё”лҰҝ м»Ён…Ңмқҙл„Ҳ <- н•„н„° <- м„ңлё”лҰҝ <- мқён„°м…үн„° <- м»ЁнҠёлЎӨлҹ¬(мҳҲмҷёл°ңмғқ)

  • мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ң л°ңмғқн•ң мҳҲмҷёлҘј мһЎм§Җ лӘ»н•ҳл©ҙ, мҳҲмҷёлҠ” м„ңлё”лҰҝ м»Ён…Ңмқҙл„Ҳм—җк№Ңм§Җ м „лӢ¬лҗңлӢӨ.
  • м„ңлё”лҰҝ м»Ён…Ңмқҙл„ҲлҠ” л„ҳм–ҙмҳЁ мҳҲмҷёлҘј кё°мӨҖмңјлЎң л“ұлЎқлҗң мҳӨлҘҳнҺҳмқҙм§ҖлҘј мЎ°нҡҢн•ңлӢӨ. л“ұлЎқлҗң кІғмқҙ м—ҶлӢӨл©ҙ кё°ліёмңјлЎң л“ұлЎқлҗҳм–ҙмһҲлҠ” 500 Internal Server Error нҺҳмқҙм§ҖлҘј нҒҙлқјмқҙм–ёнҠёлЎң л°ҳнҷҳн•ңлӢӨ.

response.sendError()

WAS <- м„ңлё”лҰҝ м»Ён…Ңмқҙл„Ҳ(sendError нҳём¶ң кё°лЎқ нҷ•мқё) <- н•„н„° <- м„ңлё”лҰҝ <- мқён„°м…үн„° <- м»ЁнҠёлЎӨлҹ¬(мҳҲмҷёл°ңмғқ)

  • response.sendError()лҘј нҳём¶ңн•ҳл©ҙ response лӮҙл¶Җм—җлҠ” мҳӨлҘҳк°Җ л°ңмғқн–ҲлӢӨлҠ” мғҒнғңлҘј м ҖмһҘн•ҳкі , м„ңлё”лҰҝ м»Ён…Ңмқҙл„ҲлҠ” кі к°қм—җкІҢ мқ‘лӢөн•ҳкё° м „м—җ responseм—җ sendError()к°Җ нҳём¶ңлҗҳм—ҲлҠ”м§Җ нҷ•мқён•ңлӢӨ. нҳём¶ңлҗҳм—ҲлӢӨл©ҙ м„Өм •н•ң HTTP мҳӨлҘҳмҪ”л“ңм—җ л§һм¶”м–ҙ мҳӨлҘҳ нҺҳмқҙм§ҖлҘј ліҙм—¬мӨҖлӢӨ.


м„ңлё”лҰҝ мҳӨлҘҳнҷ”л©ҙ л“ұлЎқ

мҠӨн”„л§Ғ л¶ҖнҠём—җм„ң м„ңлё”лҰҝ м»Ён…Ңмқҙл„ҲлҠ” мҠӨлёҢл§Ғ л¶ҖнҠёлҘј нҶөн•ҙ мӢӨн–үлҗҳлҜҖлЎң мҠӨн”„л§Ғ л¶ҖнҠёк°Җ м ңкіөн•ҳлҠ” кё°лҠҘмқ„ мӮ¬мҡ©н•ҳм—¬ м„ңлё”лҰҝ мҳӨлҘҳ нҺҳмқҙм§ҖлҘј л“ұлЎқн•  мҲҳ мһҲлӢӨ.

  • ErrorPage()лҠ” HTTP мғҒнғңмҪ”л“ң лҳҗлҠ” ExceptionлҘј нҶөн•ҙ мғқм„ұн•  мҲҳ мһҲлӢӨ.
  • Exceptionмқ„ л“ұлЎқн•  л•ҢлҠ”, мһҗмӢқ нғҖмһ…мқҳ мҳӨлҘҳлҸ„ н•Ёк»ҳ мІҳлҰ¬н•ңлӢӨ. ExceptionмқҖ лӘЁл“  мҳҲмҷёмқҳ мЎ°мғҒмқҙлҜҖлЎң мң„ мҪ”л“ңм—җм„ңлҠ” лӘЁл“  мҳҲмҷёк°Җ мһЎнһҢлӢӨ.

м„ңлё”лҰҝ м»Ён…Ңмқҙл„ҲлҠ” мҳӨлҘҳнҺҳмқҙм§ҖлҘј нҳём¶ңн•ҳкё° м „м—җ request attributeм—җ мҳӨлҘҳм—җ кҙҖн•ң м •ліҙлҘј лӢҙм•„мӨҖлӢӨ. к·ё м •ліҙлҠ” лӢӨмқҢкіј к°ҷлӢӨ.

  • javax.servlet.error.exception : мҳҲмҷё
  • javax.servlet.error.exception_type : мҳҲмҷё нғҖмһ…
  • javax.servlet.error.message : мҳӨлҘҳ л©”мӢңм§Җ
  • javax.servlet.error.request_uri : нҒҙлқјмқҙм–ёнҠё мҡ”мІӯ URI
  • javax.servlet.error.servlet_name : мҳӨлҘҳк°Җ л°ңмғқн•ң м„ңлё”лҰҝ мқҙлҰ„
  • javax.servlet.error.status_code : HTTP мғҒнғң мҪ”л“ң


DispatcherType : мҳӨлҘҳмӢң н•„н„° л„ҳм–ҙк°Җкё°

м„ңлё”лҰҝ м»Ён…Ңмқҙл„ҲлЎң л„ҳм–ҙмҳЁ м—җлҹ¬лҠ” лӢӨмқҢкіј к°ҷмқҖ кіјм •мқ„ кұ°м№ңлӢӨ

  1. м„ңлё”лҰҝ м»Ён…Ңмқҙл„Ҳ <- н•„н„° <- м„ңлё”лҰҝ <- мқён„°м…үн„° <- м»ЁнҠёлЎӨлҹ¬(мҳҲмҷёл°ңмғқ)

  2. м„ңлё”лҰҝ м»Ён…Ңмқҙл„Ҳ -> н•„н„° -> м„ңлё”лҰҝ -> мқён„°м…үн„° -> м»ЁнҠёлЎӨлҹ¬(error-404) -> view

н•ҳм§Җл§Ң н•„н„°лҘј мҳӨлҘҳнҷ”л©ҙмқ„ к°Җм ёмҳӨлҠ” кіјм •м—җм„ңк№Ңм§Җ л°ҳліөн•  н•„мҡ”лҠ” м—Ҷмқ„ кІғмқҙлӢӨ. м„ңлё”лҰҝмқҖ мқҙлҹ° л¬ём ңлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ DispatcherTypeмқ„ м ңкіөн•ңлӢӨ.

DispatcherTypeмқҖ requestм—җ л“Өм–ҙмһҲлҠ” м •ліҙлЎң request.getDispatcherType()мңјлЎң мЎ°нҡҢк°ҖлҠҘн•ҳлӢӨ. DispatcherTypeм—җлҠ” лӢӨмқҢкіј к°ҷмқҖ кІғл“Өмқҙ мһҲлӢӨ.

  • FORWARD : м„ңлё”лҰҝм—җм„ң лӢӨлҘё м„ңлё”лҰҝмқҙлӮҳ JSPлҘј нҳём¶ңн•  л•Ң

    RequestDispatcher.forward(request, response)

  • INCLUDE : м„ңлё”лҰҝм—җм„ң лӢӨлҘё м„ңлё”лҰҝмқҙлӮҳ JSPмқҳ кІ°кіјлҘј нҸ¬н•Ён•  л•Ң

    RequestDispatcher.include(request, response)

  • REQUEST : нҒҙлқјмқҙм–ёнҠё мҡ”мІӯ

  • ASYNC : м„ңлё”лҰҝ л№„лҸҷкё° нҳём¶ң

  • ERROR : мҳӨлҘҳ мҡ”мІӯ

мқҙ DispatcherTypeмқ„ нҷңмҡ©н•ҳм—¬ н•„н„°лҘј л“ұлЎқн•  л•Ң DispatcherType м„Өм •мқ„ м¶”к°Җн•ҙмЈјл©ҙ мӣҗн•ҳлҠ” DispatcherTypeмқј л•Ңл§Ң н•„н„°лҘј нҳём¶ңн•ҳлҸ„лЎқ м„Өм •к°ҖлҠҘн•ҳлӢӨ

мҠӨн”„л§Ғл¶ҖнҠё мҳҲмҷёмІҳлҰ¬

м„ңлё”лҰҝм—җ мҳӨлҘҳнҺҳмқҙм§ҖлҘј л“ұлЎқн•  л•ҢлҠ” лӢӨмқҢкіј к°ҷмқҖ кіјм •мқ„ кұ°міӨлӢӨ.

  • WebServerCustomizerлҘј л§Ңл“Ұ
  • мҳҲмҷё мў…лҘҳм—җ л”°лқј ErrorPageлҘј м¶”к°Җн•Ё
  • мҳҲмҷё мІҳлҰ¬мҡ© м»ЁнҠёлЎӨлҹ¬лҘј л§Ңл“Ұ

н•ҳм§Җл§Ң мҠӨн”„л§Ғ л¶ҖнҠём—җм„ңлҠ” мң„мҷҖ к°ҷмқҖ кіјм •мқҙ н•„мҡ”м—ҶлӢӨ.

мҳӨлҘҳнҺҳмқҙм§Җ л“ұлЎқ

мҠӨн”„л§Ғл¶ҖнҠём—җм„ңлҠ” м•„лһҳлҘј кё°ліёмңјлЎң м ңкіөн•ңлӢӨ.

  • ErrorPageлҘј мһҗлҸҷмңјлЎң л“ұлЎқн•ңлӢӨ. /errorлқјлҠ” кІҪлЎңлЎң кё°ліё мҳӨлҘҳнҺҳмқҙм§ҖлҘј м„Өм •н•ҳкі , мғҒнғңмҪ”л“ңлӮҳ мҳҲмҷёлҘј м„Өм •н•ҳм§Җ м•Ҡмңјл©ҙ кё°ліё мҳӨлҘҳнҺҳмқҙм§ҖлЎң мӮ¬мҡ©н•ңлӢӨ.
    • м„ңлё”лҰҝ л°–мңјлЎң м—җмҷёк°Җ л°ңмғқн•ҳкұ°лӮҳ, response.sendError()к°Җ нҳём¶ңлҗҳл©ҙ лӘЁл“  мҳӨлҘҳлҠ” /errorлҘј нҳём¶ңн•ңлӢӨ.
    • ErrorMvcAutoConfiguration лқјлҠ” нҒҙлһҳмҠӨк°Җ мҳӨлҘҳ нҺҳмқҙм§ҖлҘј мһҗлҸҷмңјлЎң л“ұлЎқн•ҳлҠ” м—ӯн• мқ„ н•ңлӢӨ.
  • BasicErrorControllerлқјлҠ” мҠӨн”„л§Ғ м»ЁнҠёлЎӨлҹ¬лҘј мһҗлҸҷмңјлЎң л“ұлЎқн•ңлӢӨ.
    • ErrorPageм—җм„ң л“ұлЎқн•ң /errorлҘј л§Өн•‘н•ҙм„ң мІҳлҰ¬н•ҳлҠ” м»ЁнҠёлЎӨлҹ¬лӢӨ

л”°лқјм„ң мҠӨн”„л§Ғл¶ҖнҠём—җм„ңлҠ” /error кІҪлЎңм—җ мҳӨлҘҳ нҺҳмқҙм§Җ нҢҢмқјмқ„ л§Ңл“Өм–ҙм„ң л„Јм–ҙл‘җкё°л§Ң н•ҳл©ҙ м•Ңм•„м„ң мҳӨлҘҳнҺҳмқҙм§Җк°Җ нҒҙлқјмқҙм–ёнҠёлЎң м „лӢ¬лҗңлӢӨ.мқҙл•Ң л·° м„ нғқмқҖ BasicErrorControllerм—җм„ң мқјм–ҙлӮҳкі , м•„лһҳмҷҖ к°ҷмқҖ мҡ°м„ мҲңмң„лЎң мқјм–ҙлӮңлӢӨ.
